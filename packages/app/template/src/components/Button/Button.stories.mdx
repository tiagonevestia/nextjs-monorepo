import { Button } from './Button'

# Button

With `MDX` we can define a story for `Button` right in the middle of our
Markdown documentation.

<Canvas>
  <Story
    name='Regular'
    args={{
      label: 'This is a Button',
      onClick: action('clicked')
    }}
  >
    {args => <Button {...args} />}
  </Story>
</Canvas>
